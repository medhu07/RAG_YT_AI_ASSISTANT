# Multilingual RAG-Powered YouTube AI Assistant

A Retrieval-Augmented Generation (RAG) chatbot that answers questions from YouTube video transcripts using LangChain, FAISS, Groq Llama 3.1, and Streamlit.

## Features

- YouTube transcript extraction
- English and Hindi support
- Semantic retrieval with FAISS
- RAG-based question answering
- Streamlit chat interface
- Groq-hosted Llama 3.1

## Tech Stack

- LangChain
- FAISS
- Groq
- Hugging Face Embeddings
- Streamlit
- YouTube Transcript API
